# 说人话(Say Human)

一个**作用于其它 Skill 的 Skill**。它帮你把一个装好的 Skill 翻译成人话:看懂它干嘛、怎么用,并在你真正运行它时**全程看得见每一步、在关键处随时插手**。

就像有人写了个"安全审查 Skill"用来审查别的 Skill 有没有风险——「说人话」也是一个作用于 Skill 的 Skill,只不过它管的是"看懂 + 上手 + 监督 + 调整"。

## 这是第几版

**0 版(最简 Skill 版)**,只为验证一件事:

> "在运行 Skill 时能看见过程、能在岔路口插手",到底给不给你**掌控感**?

如果这个需求确实成立、确实有用,后面再做成本地页面 / 桌面 App。

## 怎么安装

把整个 `say-human/` 文件夹放进 `~/.claude/skills/`(和你装别的 Skill 一样)。

## 怎么用

直接对 Claude 说:

- **看懂一个 Skill**:`用说人话讲讲 guizang 是干嘛的`
- **监督着跑一个 Skill**:`用说人话运行一下 guizang`
  → 它会先**解释**这个 Skill,再**问你要输入**,然后**边跑边带你**,到关键岔路口停下来让你选。一条流程走到底,你随时能停、能改、能跳。

## 它不做什么

不是 Skill 市场(找)、不是安装管理器(装)、不是给开发者看日志的工具(调试)。它只管一件事:**让你看懂、上手、监督、调整一个 Skill。**

## 一句话设计理念

掌控感来自盯住少数几个**"岔路口"**,而不是盯每一个步骤。
步骤会随 AI 越来越自主而爆炸,但真正会让结果跑偏的岔路口永远就三五个——这就是为什么我们不做"全功能监督台",只在关键决策上停下来问你。
