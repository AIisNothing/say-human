# 🟢 说人话｜guizang(社交卡片)

## 这个 Skill 是干嘛的?
把一篇文章 / 一段文字,做成**小红书图文**或**公众号封面**那种排版精致的图。

## 适合什么场景
- 小红书 3:4 封面 + 内页;公众号 21:9 + 1:1 封面
- 职场、旅行、AI/产品、读书笔记这类内容

## 它做不了什么
- 美食摆盘大片、日常 OOTD 全身照、氛围感装饰风 —— 它自己 SKILL.md 里明说做不了(`category-cookbook` 那节),会直接拒绝,不硬凑。

## 你需要给它什么
- **一段文字 / 一篇文章**(必需)
- 配图(可选,见下面的岔路口)

## 它大概会怎么做
1. `[原始]` 判断内容在不在能力圈 —— 为什么:它有些题材做不好,先拦一道
2. `[原始]` 把文章提炼成"一页一观点" —— 为什么:封面/内页要的是钩子,不是塞满字
3. `[原始]` 选风格 + 配色 + 版式 —— 为什么:决定整组图长什么样
4. `[原始]` 套模板 → 渲染成 PNG —— 为什么:出真图
5. `[增强]` 我额外加的检查点:在"配图、风格"这两处停下来问你

## ⚠️ 风险提示
- 它会**读写本地文件、装 Playwright、跑脚本渲染图片**。跑之前我会提醒你确认。
